data Source t Source #

Source type. A Source t Interface returns values of type t via its $sel:peek:Source method and flow control through its $sel:canPeek:Source and $sel:consume:Source methods.

Constructors

Source 

Fields

  • canPeek :: Bit 1

    Checks whether a value can be $sel:peek:Sourceed or $sel:consume:Sourced on this cycle.

  • peek :: t

    Returns the value that the Source is producing this cycle. The value is only relevant on a cycle where $sel:canPeek:Source is true.

  • consume :: Action ()

    Consumes a value from the Source. $sel:consume:Source should only be called on a cycle where $sel:canPeek:Source is true. When implementing a Module returning a Source Interface, one might consider using the signal returned by $sel:canPeek:Source to protect the Action with a when.

data Sink t Source #

Sink type. A Sink t Interface can be fed values of type t via its $sel:put:Sink method, expected to be called when its $sel:canPut:Sink method returns true.
